📚 Sobre o Repositório
-
Este repositório foi criado com o objetivo de armazenar e organizar projetos, atividades práticas e exercícios desenvolvidos ao longo da disciplina de Front-End cursada na faculdade.

Aqui você encontrará desde atividades realizadas em sala de aula até projetos práticos, todos focados no aprendizado e evolução nas tecnologias fundamentais do desenvolvimento front-end.

🎯 Objetivos
-
Consolidar os conhecimentos adquiridos na disciplina de Front-End

Centralizar atividades, exercícios e projetos acadêmicos

Acompanhar a evolução técnica ao longo do curso

Servir como base de estudos e portfólio futuro
